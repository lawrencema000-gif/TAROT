| Column | Purpose |
|--------|---------|
| Name | Creator name |
| Platform | IG/TikTok/YouTube |
| Followers | Count |
| Niche | Tarot/astrology/wellness |

| Response | Yes/No/Pending |
| Deal | What was agreed |
| Posted | Date of post |
| Result | Downloads/signups from their link |

## Tips

1. **Personalize every message** — reference specific content they made
2. **Don't mass DM** — 5–10 personalized outreaches per day max
3. **Lead with value** — the free premium account, not what you want from them
4. **Don't be pushy** — one follow-up max, then move on
5. **Engage first** — comment on their posts for a week before DMing
6. **Track everything** — know what works and double down
